Your Tech Stack

This architecture only includes what you used: Frontend — HTML/CSS; Backend — Python (Flask or your preferred framework); Local question generator & storage.

Frontend (HTML/CSS)

UI, timer, option selection, saving progress to localStorage, fetch questions via /api/generate-questions.

Backend (Python)

Flask server handles question generation, JSON API responses, validation, caching (optional).

Local Storage / Persistence

Store scores and answers in localStorage; optional history storage in JSON or SQLite file.

Rules & Regulations

  • Name field cannot contain numbers or special characters.
  • You must select an option to submit each question.
  • If timer runs out, the next question will automatically appear.
  • Feedback will only be displayed at the end of the quiz.
